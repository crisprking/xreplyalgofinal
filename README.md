## üöÄ Getting Started

### Prerequisites
- Node.js (v18+)
- Gemini API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd xreplyalgofinal
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure API Key:**
   - Create a `.env.local` file
   - Add your Gemini API key:
     ```
     API_KEY=your_gemini_api_key_here
     ```

4. **Run the application:**
   ```bash
   npm run dev
   ```

5. **Open in browser:**
   Navigate to `http://localhost:5173`
